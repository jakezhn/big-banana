## 品牌定位

**Bitpunk** 是一个面向系统化交易者的 AI 交易智能产品品牌。

它不强调“预测市场”或“保证盈利”，而强调：

> **确认信号、结构化分析、可审阅计划。**
> 
> 

核心定位：

**AI\-powered trading intelligence for systematic traders\.**

中文表达：

**面向系统化交易者的 AI 信号分析与交易计划审阅系统。**

---

## 品牌核心

Bitpunk 的品牌精神来自两个词：

**Bit**
代表数据、代码、系统、信号、市场结构。

**Punk**
代表反黑箱、反噪声、反投顾话术、反金融玄学。

品牌态度：

> 不相信神谕，只相信可验证的信号、结构、概率和风险边界。
> 
> 

---

## 核心设计方向

Bitpunk 的视觉不应是普通 fintech，也不应是廉价 cyberpunk。

它应该是：

> **黑色交易智能控制台 \+ 锋利 hacker\-punk 字体系统。**
> 
> 

关键词：

```Plain Text
dark
sharp
technical
rebellious
systematic
auditable
anti-hype
```

避免：

```Plain Text
币圈暴富感
霓虹城市
火箭金币
随机 glitch
粗糙像素化
游戏厅复古感
```

## Logo 设计

### 主 Logo

主 logo 使用纯文字：

```Plain Text
bitpunk.
```

设计原则：

- 首字母使用小写 **b**

- 整体为自定义锋利字体

- 不再使用粗糙像素块

- 保留张扬、尖锐、锋芒毕露的轮廓

- 字形应具备刀锋感、斜切感、攻击性和速度感

- 字体需要足够清晰，不能牺牲识别性

视觉特征：

```Plain Text
sharp cuts
blade terminals
angular counters
forward tension
hard edges
compressed energy
```

### 色彩点缀

保留两个关键识别点：

- **i 上方点：cyan**

- **k 后句点：red**

这两个点是 Bitpunk 的品牌记忆点。

## 字体与排版

### Logo 字体

自定义 display type。

风格：锋利、斜切、未来感、hacker\-punk。

不使用：

- 传统像素字体

- 圆润科技字体

- 随机 glitch 字体

- 过度游戏化字体

### UI 字体

建议组合：

```Plain Text
Primary UI: Sora / Inter / Space Grotesk
Data Mono: IBM Plex Mono / JetBrains Mono
```

排版原则：

- 标题可以锋利、强烈

- 正文必须清晰、冷静

- 数字、信号、时间戳使用 monospace

- UI 不要全站使用 logo 字体，否则会疲劳

## 品牌边界

Bitpunk 适合用于：

```Plain Text
TradingView indicator
AI trading plan workspace
signal engine
market scanner
backtest system
webhook event pipeline
trading content brand
```

不适合用于：

```Plain Text
泛企业服务
文化内容公司主体
美甲 / 私域运营业务
普通 AI 数据分析咨询
国内中性公司品牌
```

因此推荐品牌架构：

```Plain Text
MidasLab
└── Bitpunk
```

对外表达：

```Plain Text
Bitpunk
A MidasLab product
```

## 最终设计原则

Bitpunk 的设计核心不是“像素”，而是：

> **锋利、张扬、系统化、反黑箱。**
> 
> 

最终视觉应该让用户感到：

```Plain Text
这是一个交易者会信任的黑色信号工程系统。
它不是喊单工具。
它不是金融玄学。
它是可验证、可审阅、可执行的交易智能产品。
```
